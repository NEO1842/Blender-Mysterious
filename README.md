# Blender Mysterious

This is a Blender scene that creates a mysterious abstract particle visualization.

Thousands of glowing particles form flowing shapes, where blue and orange lights interact to produce a dynamic and energetic motion.

## -Test Environment-

Blender Version: 3.4
Rendering Engine: Eevee

## -How to Use-

　1. Download and open Mysterious.blend

　2. Play the animation to see the particle motion.

　3. Render the animation as MP4 (4K recommended).

## -Rendering Settings-

Rendering Engine: Eevee

Sampling: 64
Bloom: Enabled

Resolution: 4096×4096 (4K)
Output Format: MP4 (H.264)

## -Features-

　* Particle-based abstract visualization
　* Emission glow effect
　* 4K MP4 output support
　* Lightweight and real-time rendering with Eevee

# Blenderミステリアス

これは、神秘的な抽象パーティクルビジュアルを作るBlenderシーンです。

数千の発光粒子が流れるような形状を作り、
青とオレンジの光が相互作用してエネルギーのような動きを表現します。

# -テスト環境-

Blenderバージョン：3.4　レンダリングエンジン：Eevee

# -使用方法-

Mysterious.blendをダウンロードして開いてください。

1. アニメーションを再生して、パーティクルの動きをご覧ください。

2. アニメーションをMP4形式でレンダリングしてください（4K推奨）。

# -レンダリング設定-

レンダリングエンジン：Eevee

サンプリング: 64 ブルーム: 有効

解像度：4096×4096（4K）出力フォーマット：MP4（H.264）

# -特徴-

粒子ベースの抽象ビジュアル
発光グローエフェクト
4K MP4出力に対応
Eeveeによる軽量かつリアルタイムなレンダリング
